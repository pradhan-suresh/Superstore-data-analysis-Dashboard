# 📊 Super Store Sales Data Analysis

## 📌 Project Overview

The Super Store Sales Data Analysis project is an interactive business intelligence solution designed to analyze retail sales performance, customer behavior, and product profitability. Using historical sales data, the dashboard provides valuable insights that help identify sales trends, monitor business performance, and support data-driven decision-making.

The project focuses on key business metrics such as sales, profit, quantity sold, discounts, customer segments, shipping performance, and regional analysis.

---

## 🎯 Objectives

- Analyze overall sales and profit performance.
- Identify top-performing products and categories.
- Understand customer purchasing behavior.
- Evaluate regional and state-wise sales performance.
- Monitor shipping efficiency and delivery modes.
- Discover loss-making products and regions.
- Support strategic business decisions through interactive dashboards.
